### 📋 功能说明

1. **上传简历**
   - 支持格式: PDF（文本/图片）、Word文档（.doc, .docx）
   - 文件大小限制: 10MB
   - 支持拖拽上传和点击上传

2. **自动解析**
   - 文本PDF/Word: 直接提取文本
   - 图片PDF: 自动使用OCR识别
   - 提取信息: 姓名、性别、年龄、学历、学校、专业、工作经历等

3. **数据管理**
   - 查看简历列表
   - 搜索和筛选
   - 编辑和删除
   - 批量导出

### 🛑 停止应用

如果需要停止应用，可以：

1. **在终端中按 `Ctrl+C`**（如果在前台运行）
2. **查找并结束进程**:
   ```bash
   # 查找进程
   netstat -ano | findstr :5000
   
   # 结束进程（替换PID为实际进程ID）
   taskkill /PID <进程ID> /F
   ```

### 📝 注意事项

1. **首次使用OCR**:
   - 模型文件已下载，无需再次下载
   - 识别速度取决于CPU性能

2. **文件上传**:
   - 确保文件格式正确
   - 图片PDF会自动使用OCR识别

3. **数据存储**:
   - 所有数据存储在本地 `database.db`
   - 上传的文件存储在 `uploads/` 目录

### 🆘 常见问题

**Q: 应用无法启动？**
- 检查端口5000是否被占用
- 确认所有依赖已正确安装
- 查看错误日志

**Q: OCR识别失败？**
- 确认PDF图片清晰
- 检查网络连接（首次使用需要下载模型）
- 查看控制台错误信息

**Q: 如何重启应用？**
- 停止当前进程
- 重新运行 `python app.py`
