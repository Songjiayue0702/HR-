═══════════════════════════════════════════════════════════
   问题：Python未安装或未添加到系统PATH
═══════════════════════════════════════════════════════════

【第一步：检查Python是否已安装】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  双击运行：检查Python环境.bat
  
  这个脚本会检查：
  - Python是否已安装
  - Python是否添加到PATH
  - 常见的Python安装位置

【第二步：根据检查结果选择方案】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

情况A：如果Python已安装但未添加到PATH
──────────────────────────────────────────────────────────
  解决方法：
  1. 找到Python安装目录（检查脚本会显示）
  2. 将Python目录添加到系统PATH环境变量
  3. 重新打开命令提示符测试

  详细步骤请查看：Python安装指南.md

情况B：如果Python确实没有安装
──────────────────────────────────────────────────────────
  解决方法1：从官网下载安装（推荐）
  ────────────────────────────────────────────────────────
    1. 访问：https://www.python.org/downloads/
    2. 下载最新版本（Python 3.10或3.11）
    3. 运行安装程序
    4. ✅ 重要：勾选 "Add Python to PATH"
    5. 点击"Install Now"
    6. 等待安装完成
    7. 重新打开命令提示符测试

  解决方法2：从Microsoft Store安装（最简单）
  ────────────────────────────────────────────────────────
    1. 打开Microsoft Store（Microsoft 应用商店）
    2. 搜索"Python"
    3. 选择"Python 3.11"或"Python 3.10"
    4. 点击"获取"或"安装"
    5. 安装完成后会自动添加到PATH

【第三步：验证安装】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 重新打开命令提示符（必须重新打开！）
  2. 运行以下命令验证：
     
     python --version
     
     应该显示版本号，例如：Python 3.11.7

  3. 测试pip：
     
     pip --version
     
     应该显示pip版本信息

【第四步：安装项目依赖】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  在命令提示符中运行：
  
  cd "C:\Users\PC\Desktop\简历上传"
  pip install -r requirements.txt

【第五步：启动程序】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  双击运行：START_HERE.bat
  
  或手动运行：
  python app.py

═══════════════════════════════════════════════════════════
   重要提示
═══════════════════════════════════════════════════════════

1. 安装Python后，必须重新打开命令提示符才能生效
2. 如果使用安装程序安装，一定要勾选"Add Python to PATH"
3. 从Microsoft Store安装会自动添加到PATH，最简单
4. 详细说明请查看：Python安装指南.md
