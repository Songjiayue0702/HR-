手动执行步骤（在 PowerShell 中）：
═══════════════════════════════════════════════════════

1. 打开 PowerShell
   - 按 Win+X，选择 "Windows PowerShell"
   - 或按 Win+R，输入 powershell，回车

2. 切换到项目目录：
   cd "C:\Users\PC\Desktop\简历上传"

3. 添加 SSH 密钥到 ssh-agent（会提示输入密码）：
   ssh-add "$env:USERPROFILE\.ssh\id_ed25519"
   
   注意：输入密码时不会显示任何字符，这是正常的！
   直接输入密码后按 Enter 键。

4. 验证密钥已添加：
   ssh-add -l
   
   应该看到类似输出：
   256 SHA256:hMMjFPOhcINufN98RmIpO7swZiNrlzWB4V4/KLSlv6c 1057177609@qq.com (ED25519)

5. 测试 SSH 连接：
   ssh -T git@github.com
   
   应该看到类似输出：
   Hi Songjiayue0702! You've successfully authenticated...

6. 推送代码到 GitHub：
   git push origin main
