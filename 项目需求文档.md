# 1. 项目概述

## 1.1. 项目背景

随着企业招聘流程的规范化与效率化，HR部门需要处理大量来自不同渠道的简历。手动从海量、格式各异的简历中提取关键信息，不仅耗时耗力，而且容易出错。本项目旨在开发一个基于网页的智能简历数据库系统，通过自动化解析技术，快速、准确地从上传的简历文件中提取结构化信息，并提供便捷的查阅、管理和导出功能，极大提升招聘前期的简历筛选与人才库管理效率。



## 1.2. 项目目标

核心目标： 实现一个用户友好、准确高效的在线简历信息提取与管理平台，通过AI API智能识别技术确保关键实体信息的准确性和标准化。



关键指标：



支持主简历格式（PDF文字、PDF图片、Word文档）的解析



对指定字段（姓名、性别、年龄、工龄等）的提取准确率 > 85%



新增： 通过AI API对公司名称、学校名称的智能识别率 > 90%



提供直观的数据展示界面和灵活的数据导出功能



系统响应快速，考虑到API调用，文件上传和解析平均耗时 < 60秒



## 1.3. 项目范围

范围内：

简历文件上传、存储和管理



文本提取和关键信息解析（通过AI API智能识别）



AI API集成（统一使用AI API进行公司、学校、专业智能识别）



简历数据的展示、搜索、筛选



单个和批量数据导出功能



基础的数据编辑和修正功能



范围外：

候选人面试安排和流程管理



招聘岗位发布和管理



多级审批工作流



移动端APP开发



高级数据分析报表

# 3. 功能需求

## 3.1. 核心功能模块

### 模块一：简历上传与解析

#### 1. 文件上传

功能描述： 提供友好、高效的文件上传界面



需求详情：



支持拖拽上传和点击上传两种方式



支持格式：.pdf, .doc, .docx



支持单文件上传和批量文件上传（如ZIP压缩包）



上传时显示实时进度条和状态提示



文件格式和大小校验（限制为10MB）



重复文件检测和提示



上传中断支持断点续传



#### 2. 智能解析（增强）

功能描述： 自动解析简历内容并提取关键信息，统一使用AI API进行智能识别和验证



触发条件： 文件上传成功后自动触发解析流程



解析能力：



文本PDF/Word： 直接提取文本内容，通过AI API进行智能分析



图片PDF： 通过AI API直接处理图片PDF，AI API支持图片识别，无需OCR引擎



解析字段详情：



字段	说明	逻辑/规则（更新）	API集成

姓名	候选人的全名	通常位于简历顶部，通过命名实体识别技术提取	无

性别	男 / 女	通过关键词识别（如"男"、"女"）或分析代词	无

出生年份	年份数字	通过识别"出生"、"生日"等关键词后的日期信息提取	无

年龄	整数	计算字段：当前年份 - 出生年份	无

最早工作年份	年份数字	分析所有工作经历，找出最早的开始年份	无

工龄	整数	计算字段：当前年份 - 最早工作年份	无

工作经历	列表	【增强】 提取每段经历的公司名称和岗位名称。通过AI API智能识别公司名称，返回准确的全称	AI API

最高学历	枚举值	按预设优先级（博士>硕士>本科>专科&大专>高中>职高>初中）匹配，通过AI API智能识别	AI API

学校	字符串	【增强】 提取最高学历对应的院校名称。通过AI API智能识别，返回准确的学校全称，并区分"本科院校"与"专科院校"等	AI API

专业	字符串	【增强】 提取最高学历对应的专业名称。通过AI API智能识别，返回标准的专业名称	AI API

AI识别状态管理：



识别成功：AI API返回准确且置信度高的结果



需确认：AI API返回多个可能结果或置信度中等，系统应记录并允许后续手动选择



识别失败：AI API未返回结果或置信度过低



未识别：因网络或API限制等原因未调用AI识别



数据存储策略：



数据库同时存储原始文本和AI识别后的文本



记录AI API调用时间、识别状态和置信度



解析结果处理：



解析成功后，数据存入数据库，并在界面上提示"解析成功"



对于识别状态非"识别成功"的字段，应进行高亮或标记，提示用户关注



解析失败时，在界面上明确提示"解析失败"，并允许用户手动录入或重新上传



### 模块二：简历数据管理（增强）

#### 1. 数据看板/列表

功能描述： 提供清晰、直观的简历数据概览和快速操作入口



需求详情：



以表格形式展示所有已成功解析的简历信息



显示核心字段：姓名、性别、年龄、工龄、最高学历、学校、专业



支持按所有字段进行升序/降序排序



提供强大的筛选功能：



文本搜索：姓名、学校、专业



范围筛选：年龄范围、工龄范围



枚举筛选：性别、最高学历



AI识别状态筛选：公司识别状态、学校识别状态



分页显示，支持自定义每页显示数量



快速预览功能（鼠标悬停显示简要信息）



#### 2. 详情查看与纠偏（增强）

功能描述： 提供详细的候选人信息展示和交互式修正功能



需求详情：



点击列表中的任一候选人姓名或"查看详情"按钮，弹出模态框或跳转至详情页



详情页展示该候选人所有被解析出的完整信息



【新增】 AI识别状态可视化：



绿色对勾图标✅表示"识别成功"



黄色问号图标❓表示"需确认"



红色叉号图标❌表示"识别失败"



灰色图标表示"未识别"



【新增】 交互式纠偏功能：



点击识别状态不佳的字段，弹出修正面板



对于"需确认"状态，显示AI返回的备选列表供用户选择



对于"识别失败"状态，提供手动输入和AI重新识别功能



手动输入时提供自动完成建议



工作经历时间轴可视化展示



教育背景详细信息展示



#### 3. 数据编辑与补全

功能描述： 允许用户手动修正和完善解析结果



需求详情：



在详情页面，允许用户对任何解析错误的字段进行手动编辑



提供富文本编辑器用于复杂内容的编辑



所有修改实时保存，提供撤销/重做功能



编辑历史记录和版本对比



批量编辑功能（如批量修改学历信息）



### 模块三：数据导出

#### 1. 单独下载

功能描述： 导出单个候选人的完整信息



需求详情：



在简历详情页，提供"下载此简历信息"按钮



下载格式为标准的.xlsx或.csv文件



文件内容包含该候选人的完整解析信息



可选择的导出格式和字段



包含原始文本和标准化文本的对比



#### 2. 批量下载（增强）

功能描述： 批量导出选中或全部候选人的信息



需求详情：



在简历列表页面，提供复选框，允许用户勾选一个或多个候选人



提供"导出选中项"按钮，将所选候选人的所有信息导出为一个.xlsx文件



提供"导出全部"按钮，将当前列表视图下的所有候选人信息导出



【增强】 导出的Excel文件中包含：



AI识别后的公司/学校/专业名称



原始文本和识别状态标识



AI识别的置信度分数



数据解析时间戳



支持自定义导出模板



导出任务队列管理，大文件导出提供后台处理和邮件通知
