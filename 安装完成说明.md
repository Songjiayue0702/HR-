### 📝 配置说明

系统已自动配置为使用 **EasyOCR** 作为OCR引擎：

- **配置文件**: `config.py`
  ```python
  OCR_ENABLED = True
  OCR_ENGINE = 'easyocr'
  OCR_USE_GPU = False
  ```

### 🚀 使用方法

1. **启动应用**:
   ```bash
   python app.py
   ```

2. **上传图片PDF**:
   - 系统会自动检测PDF是否为图片格式
   - 如果直接提取文本失败或文本过少（<100字符），会自动使用OCR识别
   - 首次使用OCR时会自动下载模型文件（约500MB），请确保网络畅通

### ⚠️ 注意事项

1. **首次使用**:
   - 首次运行OCR时会自动下载模型文件
   - 下载大小约500MB
   - 请确保网络连接正常

2. **性能**:
   - 当前使用CPU模式（速度较慢但无需GPU）
   - 如需GPU加速，需要安装CUDA版本的PyTorch

3. **PaddleOCR**:
   - 由于Windows长路径限制，PaddleOCR安装失败
   - 已改用EasyOCR作为替代方案
   - EasyOCR对中文识别效果良好

### 🔧 如果遇到问题

1. **OCR识别失败**:
   - 检查网络连接（首次使用需要下载模型）
   - 确认PDF文件清晰度
   - 查看控制台错误信息

2. **想要使用PaddleOCR**:
   - 需要启用Windows长路径支持
   - 或使用虚拟环境重新安装

3. **想要禁用OCR**:
   - 在 `config.py` 中设置 `OCR_ENABLED = False`
