# 正确的推送命令说明

## ❌ 常见错误

### 错误 1: 环境变量语法错误

**错误的命令：**
```powershell
ssh-add "$env: USERPROFILE\.ssh\id_ed25519"  # ❌ $env: 和 USERPROFILE 之间有空格
```

**正确的命令：**
```powershell
ssh-add "$env:USERPROFILE\.ssh\id_ed25519"   # ✅ 没有空格
```

### 错误 2: 使用 ssh-ed25519 作为命令

`ssh-ed25519` 是密钥类型，不是命令！不要直接执行它。

## ✅ 正确的步骤

### 在 PowerShell 中执行：

```powershell
# 1. 切换到项目目录（如果还没有）
cd "C:\Users\PC\Desktop\简历上传"

# 2. 添加 SSH 密钥（注意：$env: 和 USERPROFILE 之间没有空格）
ssh-add "$env:USERPROFILE\.ssh\id_ed25519"

# 3. 验证密钥已添加
ssh-add -l

# 4. 推送代码
git push origin main
```

## 🔑 关于密码输入

- 输入密码时**不会显示任何字符**，这是正常的安全机制
- 直接输入密码后按 Enter 键
- 如果显示 "Bad passphrase"，说明密码输入错误
- 如果忘记密码，需要重新生成 SSH 密钥
